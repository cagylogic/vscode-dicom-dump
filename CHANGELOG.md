# Change Log

## 1.2.0

- Added new configuration `dicom.searches`.
- Fixed bugs regarding ISO-2022-JP encoding (Japanese).

- Support for (retired) Group Length element (gggg,0000).

- Documentation and changelog fix.

## 1.0.0 (2018-07-08)

- Initial release.
